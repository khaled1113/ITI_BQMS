# Pharonic QR Code Generator

This project is a simple web application that generates QR codes for users in a pharaonic-themed design. The application is hosted on GitHub Pages and allows users to easily create QR codes by visiting the website.

## Project Structure

- `src/index.html`: The main HTML document for the website.
- `src/styles/style.css`: Contains the styles for the website, reflecting a pharaonic theme.
- `src/scripts/app.js`: JavaScript code that generates the QR code using a QR code library.
- `package.json`: Configuration file for npm, listing dependencies and scripts.

## Features

- Generates QR codes based on user input.
- Pharaonic-themed design for an engaging user experience.
- Responsive layout for accessibility on various devices.

## Getting Started

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/pharonic-qr-website.git
   ```

2. Navigate to the project directory:
   ```
   cd pharonic-qr-website
   ```

3. Install the necessary dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the application.

## Usage

- Enter the data you want to encode in the QR code.
- Click the "Generate QR Code" button to create your QR code.
- The QR code will be displayed on the page for you to scan or download.
